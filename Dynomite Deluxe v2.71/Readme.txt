DynomiteSetup-en.exe -- установщик
Dynomite.exe         -- патч

Инструкция по установке Dynomite Deluxe v2.71 rev"60-minute trial"

1) Установите игру с помошью установщика
#) Рекомендуется один раз запустить игру (Но не через пункт в установщике "Launch game now" -- выйдите из установщика и запустите игру через ссылку на рабочем столе*) Не забудте выйти из игры и/или установщика
2) Нажмите правой кнопкой мыши по ярлыку игры на рабочем столе, и выберете пункт "Свойства". В окне свойств ярлыка выберте "Расположение файла" или "Найти Обект"
3) Скопируйте патч в папку которую вы только-что открыли. Файл который заменяется лутше сохранить.
4) Запускаем игру
5) Радуемся!
